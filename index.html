VSP Source Locator

This is a utility to calculate the location of a single or selection of VSP sources. It calculates source offset and azimuth from the X and Y coordniates and vice versa. The calculations are not unique to a VSP survey and the utility could be used to calculate the distance between any two fixed points.

To use input the wellhead X and Y coordinates and either the source X and Y coordinates or the source offset and azimuth (degrees clockwise from north) from wellhead. The source locations can be UTM or other local coordinates or simple offsets (from for example a wellhead at 0, 0) Click the Calculate! button to run the calculation. The Add Source button allows the calculation of source locations to generate a list of sources. The Creat Plan button generates an X, Y plan of the wellhead and source locations to allow for a QC of the calculated information.
