VSP Source Locator

Instructions

This is a utility to calculate the location of a single or selection of VSP sources. It uses either the source X and Y location to calculate offset and azimuth from wellhead or the offset and azimuth to calculate the X and Y location.

To use, input the wellhead X and Y location and either the source X and Y or offset and azimuth (degrees clockwise from north) from wellhead. The source X and Y locations can be UTM or other local coordinates or simple offsets (from for example a wellhead at 0, 0). The input source and wellhead X and Y must be in the same coordinate system. Click the Calculate! button to run the calculation. After the initial calculation the Add Source button allows the calculation and generation of a list of source locations connected to the same wellhead. The Create Plan button generates an X, Y plan of the wellhead and source locations to allow for a QC of the calculated information.
